Percepta

Python Raspberry Pi 4 Computer Vision / AI HTML / CSS Java

AI-powered wearable glasses for the visually impaired. The device uses a camera and real-time AI video analysis to describe what is in front of the user — interpreting surroundings, obstacles, and objects and delivering audio feedback. Think META smart glasses, but purpose-built for accessibility and far more affordable.

Built on a Raspberry Pi 4, Percepta processes live video frames through a computer vision model and converts scene descriptions to audio in real time. Currently in active development and in discussions for potential sponsorship.
